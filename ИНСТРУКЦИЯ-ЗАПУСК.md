# Как запустить бота — пошаговая инструкция

Это разовая настройка. После неё бот будет работать сам, круглосуточно, без твоего участия.

---

## Шаг 1. Создать бота в Telegram

1. Открой Telegram, найди аккаунт **@BotFather** (это официальный бот от самого Telegram).
2. Напиши ему `/newbot`.
3. Он спросит имя бота (то, что видят люди) — например: `Джйотиш Катя Дас`.
4. Затем спросит username бота (должен заканчиваться на `bot`) — например: `katya_das_jyotish_bot`.
5. BotFather пришлёт тебе **токен** — длинную строку вида `123456789:ABCdefGHIjklMNOpqrSTUvwxYZ`. 

**Сохрани этот токен — он понадобится на шаге 3.** Никому не показывай его, это как пароль от бота.

---

## Шаг 2. Завести хостинг (место, где бот будет работать)

Рекомендую **Railway** (railway.app) — там есть бесплатный пробный период, а дальше оплата по факту использования (для одного бота с небольшим потоком это обычно несколько долларов в месяц).

1. Зайди на **railway.app**, зарегистрируйся (можно через Google-аккаунт).
2. Нажми **New Project** → **Deploy from GitHub repo** (или **Empty Project**, если не хочешь связываться с GitHub — тогда файлы можно будет загрузить вручную через интерфейс Railway или CLI).

Если с GitHub возиться не хочется — самый простой вариант: попроси любого знакомого, кто пользуется компьютером уверенно (не обязательно программиста), помочь один раз залить папку с ботом на Railway через их простой веб-интерфейс drag-and-drop. Это займёт 10 минут.

---

## Шаг 3. Указать токен

В настройках проекта на Railway найди раздел **Variables** (переменные окружения) и добавь:

```
BOT_TOKEN = твой_токен_из_шага_1
DB_PATH = /data/janma-kundali.db
RAILWAY_RUN_UID = 0
ADMIN_ID = твой_telegram_id (узнать командой /whoami у самого бота)
```

**Про DB_PATH и Volume**: чтобы архив карт и список пользователей не стирались при каждом обновлении бота, на Railway нужно один раз создать Volume (постоянный диск) и примонтировать его в `/data` — см. раздел проекта Railway → Volumes → New Volume → Mount Path: `/data`.

---

## Шаг 4. Запуск

Railway сам определит, что это Node.js проект (по файлу `package.json`) и запустит команду `npm start`, которая написана внутри — она включает бота.

После этого в течение минуты бот должен появиться "живым" в Telegram — напиши ему `/start`.

---

## Как пользоваться ботом

- `/start` — приветствие
- `/chart` — построить карту (бот задаст вопросы по очереди: дата → время → координаты места)
- `/help` — список команд

Сейчас бот принимает координаты места напрямую (широта, долгота, часовой пояс) — автоматический подбор по названию города будет добавлен отдельным шагом позже.

---

## Если что-то не заработало

Самая частая причина — неправильно скопированный токен (лишний пробел в начале/конце). Проверь Variables на Railway и убедись, что там ровно то, что прислал BotFather.

Если возникнут сложности на любом шаге — просто пришли мне скриншот того, что видишь, и я подскажу, что делать дальше.
